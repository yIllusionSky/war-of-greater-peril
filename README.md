# EU5 Super Mod

欧陆风云5增强型平衡修改mod

## 简介

本mod旨在提升《欧陆风云5》的游戏体验。针对原游戏中的一些不合理之处进行了调整和完善，包括优化人口机制、增强建筑系统、改进经济平衡等多个方面。

## 主要特性

- **人口机制优化** - 改进人口增长和损失的平衡性
- **建筑系统增强** - 新增和改进各类建筑物
- **经济平衡调整** - 修改商品需求和建筑成本
- **修饰符改进** - 优化各类静态修饰符和位置等级

## 包含内容

- 医院建筑改进
- 人口修饰符调整
- 社会价值观系统优化
- 商品需求平衡
- 位置等级优化

## 兼容性

支持游戏版本：1.0.x 及以上
