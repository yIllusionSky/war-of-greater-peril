# 欧陆天际线

[github](https://github.com/yIllusionSky/Eurapo-Skyline)

欧陆风云5增强型平衡修改mod

## 简介

本mod旨在提升《欧陆风云5》的游戏体验。针对原版游戏中人口增长过慢的问题进行了优化，并全面增强了“种田”体验，包括优化人口机制、增强建筑系统、改进经济平衡等多个方面。

## 借用内容

本mod借用了以下mod的部分理念或内容：
- Better Hospital
- Megalopolis Upgrade
- 人口增长调整&季风

## 主要特性

- **人口机制优化** - 改进人口增长和损失的平衡性
- **建筑系统增强** - 新增和改进各类建筑物
- **经济平衡调整** - 修改商品需求和建筑成本
- **修饰符改进** - 优化各类静态修饰符和位置等级

## 包含内容

- 医院建筑改进
- 人口修饰符调整
- 社会价值观系统优化
- 商品需求平衡
- 位置等级优化

## 兼容性

支持游戏版本：1.0.x 及以上
